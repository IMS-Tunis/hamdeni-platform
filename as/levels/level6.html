1. Basic String Operations

  • Immutability: Python strings are immutable. Operations and methods return new strings and do not modify the original.
  • Concatenation: combine multiple strings using the + operator.
Example A - Concatenate strings

Concatenation with +
Length with len()
greeting = "Hello" + " " + "World"
print(greeting) # Hello World
  • String length: len(s) returns the number of Unicode code points. For most simple texts this equals the number of characters, but combining marks can make visual characters use multiple code points.
Example B - Length of a string
message = "Python"
print(len(message)) # 6

2. Accessing and Manipulating Substrings

  • Indexing: positions start at 0 for the first character.
Example A - Access characters by index

Indexing: s[i]
Slicing: s[a:b]
Negative indices
ord/chr conversions
word = "Hello"
print(word[0]) # H
print(word[4]) # o
  • Slicing: s[start:end] returns characters from start inclusive to end exclusive. Omitting start defaults to 0; omitting end defaults to len(s).
  • s[start:end:step] with default step = 1. A negative step reverses direction; example: text[::-1] produces the reversed string.
Example B - Extract a substring
phrase = "Python Programming"
print(phrase[0:6]) # Python
  • Negative indexing: -1 refers to the last element, -2 to the one before it. Slices tolerate out-of-range bounds, but single-index access that is out of range raises IndexError.
Example C - Negative indices
text = "Python"
print(text[-1]) # n
print(text[-2]) # o
  • Character and Unicode conversion: ord(c) returns the Unicode code point of a single-character string; chr(n) returns the character for an integer n.
Example D - Unicode conversions
char = 'A'
print(ord(char)) # 65
ascii_value = 65
print(chr(ascii_value)) # A

Constraints: ord expects a string of length 1 (otherwise TypeError). chr requires an integer 0..0x10FFFF (otherwise ValueError).

3. String Methods

  • Changing case: upper() and lower() convert to uppercase or lowercase.
Example A - Convert case

Case: upper()/lower()
Strip whitespace: strip()
Find vs index

Replace text
isdigit, islower, isupper
Split into words
name = "python"
print(name.upper()) # PYTHON
name = "PYTHON"
print(name.lower()) # python
  • Stripping whitespace: strip() removes leading and trailing Unicode whitespace by default; optionally pass characters to remove as an argument.
Example B - Remove outer spaces
user_input = " hello "
print(user_input.strip()) # hello
  • Finding substrings: find(sub[, start[, end]]) returns the lowest index where sub occurs or -1 if absent; index(sub[, start[, end]]) has the same search but raises ValueError if absent.

Examples: "abc".find("z") returns -1, while "abc".index("z") raises ValueError.

Example C - Locate a substring
sentence = "I love Python"
print(sentence.find("Python")) # 7
  • Replacing text: replace(old, new) returns a new string with replacements.
Example D - Replace a word
text = "I like apples"
print(text.replace("apples", "bananas")) # I like bananas

Strings are immutable. replace returns a new string. The original only changes if you assign the result to the variable, e.g., text = text.replace("a", "b").

  • Boolean tests: isdigit() is True if all characters are Unicode digits; islower() or isupper() are True if the string contains at least one cased letter and all cased letters are lower or upper respectively.
Example E - Check string properties
number = "12345"
print(number.isdigit()) # True
word = "hello"
print(word.islower()) # True
word = "HELLO"
print(word.isupper()) # True
  • Splitting text: split() without arguments divides a string into substrings on runs of whitespace and omits the separators.
Example F - Split into words
sentence = "Python is fun"
words = sentence.split()
print(words) # ['Python', 'is', 'fun']

4. String Comparison

  • Equality and inequality: equality tests are case sensitive. Ordering comparisons use lexicographic order by Unicode code point. For case-insensitive testing, normalise or use casefold() when appropriate.
Example A - Compare strings

Equality and inequality
Membership with in operator
word1 = "Python"
word2 = "python"
print(word1 == word2) # False
print(word1 != word2) # True
  • Membership with in: test if a substring occurs inside a string.

The in operator checks substrings, not whole words, and it is case sensitive. Examples: "on" in "Python" is True; "py" in "Python" is False.

Example B - Check for containment
sentence = "I love Python"
print("Python" in sentence) # True
print("Java" in sentence) # False
