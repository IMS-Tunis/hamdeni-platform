What is an Array?

  • Definition: An array is a collection of elements of the same data type, stored under one name. Each element is accessed using an index.
  • 1D array: A simple list of elements accessed by a single index.
  • 2D array: A table-like structure accessed by two indices: row and column.
  • Why use arrays? Arrays store and manage related data, such as a list of numbers or names.

In these notes, arrays are 0-based: the first element has index 0.

1. Declaring and Initializing Arrays

  • Declare a 1D array: set the index range and element type.
Example A - Declare a 1D array
DECLARE numbers : ARRAY [0:4] OF INTEGER
  • Declare a 2D array: specify row and column index ranges and the element type.
Example B - Declare a 2D array
DECLARE matrix : ARRAY [0:2, 0:2] OF INTEGER
  • Initialize arrays: assign values to elements by index.
Example C - Initialize a 1D array
numbers[0] ← 10
numbers[1] ← 20
numbers[2] ← 30
numbers[3] ← 40
numbers[4] ← 52
Example D - Initialize a 2D array
matrix[0, 0] ← 1
matrix[0, 1] ← 2
matrix[0, 2] ← 3
matrix[1, 0] ← 4
matrix[1, 1] ← 5
matrix[1, 2] ← 6
matrix[2, 0] ← 7
matrix[2, 1] ← 8
matrix[2, 2] ← 9

For 2D arrays we write matrix[row, column]. The examples above fill the matrix row by row.

2. Accessing and Modifying Array Elements

  • Access elements: use indices to read a value.
Example A - Access elements
OUTPUT numbers[2]
OUTPUT matrix[1, 1]
  • Modify elements: assign a new value to an existing position.
Example B - Modify elements
numbers[2] ← 35
matrix[1, 1] ← 50

3. Looping Through Arrays

  • Use loops: visit each element in a 1D or 2D array in order.
Example A - Loop over a 1D array
DECLARE i : INTEGER
FOR i ← 0 TO LENGTH(numbers) - 1
OUTPUT numbers[i]
NEXT i
Example B - Loop over a 2D array (row-major)
DECLARE i, j : INTEGER
FOR i ← 0 TO 2
FOR j ← 0 TO 2
OUTPUT matrix[i, j]
NEXT j
NEXT i

LENGTH(numbers) returns the number of elements. For a 0‑based array of length 5, valid indices are 0..4.

4. Records

  • What is a record? A composite data structure that groups related fields of different types under one name.
Example A - Declare a record
DECLARE Student : RECORD
name : STRING
age : INTEGER
grade : REAL
END RECORD
  • Access and modify fields: use the dot notation RecordName.field.
Example B - Use a record
Student.name ← "Alice"
Student.age ← 20
Student.grade ← 85.5
OUTPUT Student.name & " is " & Student.age & " years old with a grade of " & Student.grade
