Introduction to Strings

  • Definition: A string is a sequence of characters that can be manipulated using built‑in functions.
  • Example: "Hello World" has 11 characters (including the space).

These notes follow the Cambridge 9618 style. For substring positions, indexing starts at 1 unless stated otherwise.

Basic String Operations

  • Concatenation: join strings with &. Example: "Hello" & " " & "World" gives "Hello World".
  • Comparisons: use relational operators =, <> (not equal), >, <, >=, <= to compare strings.
Example A - Concatenation & comparison
DECLARE s1, s2 : STRING
s1 ← "Hello"
s2 ← "World"
OUTPUT s1 & " " & s2 // "Hello World"
OUTPUT ("Hi" = "Hi") // TRUE
OUTPUT ("AB" <> "BA") // TRUE

String Manipulation Functions

Function signatures and examples follow the Cambridge 9618 pseudocode reference.

LEFT(ThisString : STRING, x : INTEGER) RETURNS STRING
Returns the leftmost x characters from ThisString.
Example: LEFT("ABCDEFGH", 3) returns "ABC"
RIGHT(ThisString : STRING, x : INTEGER) RETURNS STRING
Returns the rightmost x characters from ThisString.
Example: RIGHT("ABCDEFGH", 3) returns "FGH"
MID(ThisString : STRING, x : INTEGER, y : INTEGER) RETURNS STRING
Returns a string of length y starting at position x from ThisString.
Example: MID("ABCDEFGH", 2, 3) returns "BCD"
LENGTH(ThisString : STRING) RETURNS INTEGER
Returns the number of characters in ThisString.
Example: LENGTH("Happy Days") returns 10
TO_UPPER(x : <datatype>) RETURNS <datatype>
<datatype> may be CHAR or STRING.
Converts all characters of x to upper case; returns the same type as the input.
Examples: TO_UPPER("Error 803") → "ERROR 803"; TO_UPPER('a') → 'A'
TO_LOWER(x : <datatype>) RETURNS <datatype>
<datatype> may be CHAR or STRING.
Converts all characters of x to lower case; returns the same type as the input.
Examples: TO_LOWER("JIM 803") → "jim 803"; TO_LOWER('W') → 'w'
NUM_TO_STR(x : <datatype1>) RETURNS <datatype2>
Returns a string representation of a numeric value.
<datatype1> may be REAL or INTEGER; <datatype2> may be CHAR or STRING.
If x is negative, the returned value begins with the '-' character.
Example: NUM_TO_STR(87.5) → "87.5"
STR_TO_NUM(x : <datatype1>) RETURNS <datatype2>
Returns a numeric representation of a string.
<datatype1> may be CHAR or STRING; <datatype2> may be REAL or INTEGER.
Example: STR_TO_NUM("23.45") → 23.45
If the string begins with '-', the returned value will be negative.
IS_NUM(ThisString : <datatype>) RETURNS BOOLEAN
Returns TRUE if ThisString represents a valid numeric value.
<datatype> may be CHAR or STRING.
Example: IS_NUM("-12.36") → TRUE
ASC(ThisChar : CHAR) RETURNS INTEGER
Returns the integer (ASCII) value of character ThisChar.
Example: ASC('A') → 65; ASC('B') → 66
CHR(x : INTEGER) RETURNS CHAR
Returns the character whose integer (ASCII) value is x.
Example: CHR(65) → 'A'; CHR(66) → 'B'
