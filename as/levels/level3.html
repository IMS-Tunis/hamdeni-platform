What is Iteration?

  • Definition: Iteration repeats a set of instructions multiple times. It can be controlled by a counter, a condition checked before the loop, or a condition checked after the loop.

1. FOR Loops (Count-Controlled Loop)

  • FOR loop: A count-controlled loop that runs a specific number of times, determined by a counter.
Example A - Basic FOR loop
DECLARE i : INTEGER
FOR i ← 1 TO 5
OUTPUT "This is iteration " & i
NEXT i

Use a FOR loop when you know the number of iterations in advance.

2. WHILE Loops (Pre-Condition Loop)

  • WHILE loop: Checks its condition before each iteration and runs as long as the condition is TRUE.
Example A - Basic WHILE loop
DECLARE count : INTEGER
count ← 0
WHILE count < 5
OUTPUT "Count is " & count
count ← count + 1
ENDWHILE

Use a WHILE loop when you do not know the exact number of iterations in advance.

3. REPEAT-UNTIL Loops (Post-Condition Loop)

  • REPEAT-UNTIL loop: Checks the condition after each iteration, so the loop body runs at least once.
Example A - Basic REPEAT-UNTIL loop
DECLARE count : INTEGER
count ← 0
REPEAT
OUTPUT "Count is " & count
count ← count + 1
UNTIL count = 5

In REPEAT-UNTIL, the loop continues until the condition becomes TRUE.

REPEAT-UNTIL loop (post-condition)

Runs the body at least once, then repeats until the condition is TRUE.

4. Choosing the Most Appropriate Loop

  • FOR: When the number of iterations is known beforehand.
  • WHILE: When the number of iterations depends on a condition checked before each iteration.
  • REPEAT-UNTIL: When the loop must run at least once and the stopping condition is checked after each iteration.
