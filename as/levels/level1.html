0. What is Pseudocode?

  • Definition: Pseudocode is an informal way to write programming logic without using the strict rules of a specific language.
  • Purpose: It mixes natural language with code-like structures to describe algorithms clearly.

Pseudocode focuses on the steps of the solution, not on language syntax. It should be easy to read and unambiguous.

1. Variables

  • What are variables? Named storage locations in memory for values that your algorithm uses and updates.
  • Common uses: Store numbers, text, or logical values needed by later steps.
Example A - Assign values to variables
age ← 20
name ← "Alice"

Declaration of Variables

  • What is declaration? Defining a variable by its name and data type so the algorithm knows what kind of data it will store.
  • Syntax: DECLARE <variableName> : <dataType>
  • <variableName>: the name of the variable.
  • <dataType>: the kind of data the variable will store.
Example B - Declare variables
DECLARE age : INTEGER
DECLARE name : STRING
DECLARE isStudent : BOOLEAN

2. Data Types

  • Definition: A data type defines the kind of data a variable can hold.
  • INTEGER: Whole numbers (e.g., 5, -3).
  • REAL: Numbers with decimals (e.g., 4.7, -2.0).
  • CHAR: A single character (e.g., 'A', '3').
  • STRING: A sequence of characters (e.g., "Hello", "1234").
  • BOOLEAN: Logical values TRUE or FALSE.

Choose a data type that matches the values you will store. For example, use INTEGER for counts and BOOLEAN for decision flags.

3. Basic Input and Output

  • Input: Read data from the user with INPUT.
  • Output: Display information to the user with OUTPUT.
Example A - Read input
INPUT age
INPUT name
Example B - Write output
OUTPUT "Your name is " & name
OUTPUT "Your age is " & age

4. Arithmetic Operations

  • Common operators: + (addition), - (subtraction), * (multiplication), / (real division), DIV (integer division), MOD (remainder).
  • Real division: Produces a real number even with integer inputs.
  • Integer division: DIV gives the integer quotient only.
  • Modulus: MOD gives the remainder after division.
Example A - Division and remainder
DECLARE a : INTEGER
DECLARE b : INTEGER
DECLARE resultReal : REAL
DECLARE resultInteger : INTEGER
DECLARE remainder : INTEGER
a ← 10
b ← 3
resultReal ← a / b
resultInteger ← a DIV b
remainder ← a MOD b
OUTPUT "Real Division: " & resultReal
OUTPUT "Integer Division: " & resultInteger
OUTPUT "Remainder: " & remainder

Remember: DIV removes the fractional part; MOD gives only the remainder.
