Trace the algorithm and give the final contents of the array.

FOR Counter ← 1 TO 3
  Number[Counter] ← Number[Counter] + Number[Counter - 1]
NEXT Counter

Also state the value stored in each position after every iteration.

[4, -2, 7, 1]

[4, 2, 7, 1]

After Counter = 2:
[4, 2, 9, 1]

Exercise 9 - Nested loops with a table

Study the algorithm.

FOR ColumnCounter ← 0 TO 1
  FOR RowCounter ← 0 TO 2
    INPUT MyTable[RowCounter, ColumnCounter]
  NEXT RowCounter
NEXT ColumnCounter

1) Why are two loops needed?
2) Which element is filled first?
3) Which element is filled second?
4) Which element is filled last?

Mark scheme
1) Because a two-dimensional array has two indexes, so one loop is needed for each index.
2) MyTable[0,0]
3) MyTable[1,0]
4) MyTable[2,1]

Exercise 10 - Process a one-dimensional array

Write Cambridge-style pseudocode to browse all elements of the array Number.

DECLARE Number : ARRAY[0:5] OF INTEGER

Your algorithm must:

1) count how many positive integers are stored in the array
2) calculate the sum of the positive integers
3) calculate the average of the positive integers

Assume that the array contains at least one positive integer.

Mark scheme
DECLARE Count : INTEGER
DECLARE Sum : INTEGER
DECLARE Average : REAL
DECLARE Counter : INTEGER

Count ← 0
Sum ← 0

FOR Counter ← 0 TO 5
  IF Number[Counter] > 0 THEN
    Count ← Count + 1
    Sum ← Sum + Number[Counter]
  ENDIF
NEXT Counter

Average ← Sum / Count

OUTPUT Count
OUTPUT Sum
OUTPUT Average

Exercise 11 - Replace negative values in a two-dimensional array

Write Cambridge-style pseudocode to browse all elements of the two-dimensional array Data.

DECLARE Data : ARRAY[0:2,0:3] OF INTEGER

Your algorithm must set every negative value in the array to zero.

Mark scheme
DECLARE RowCounter : INTEGER
DECLARE ColumnCounter : INTEGER

FOR RowCounter ← 0 TO 2
  FOR ColumnCounter ← 0 TO 3
    IF Data[RowCounter, ColumnCounter] < 0 THEN
      Data[RowCounter, ColumnCounter] ← 0
    ENDIF
  NEXT ColumnCounter
NEXT RowCounter
