1. Totalling

  • Totalling means adding values together to keep a running total.
  • To write a totalling program, you usually:
    • initialise the total to 0
    • input one value at a time
    • add each new value to the total
Example 1: Total 10 numbers entered by the user.
Total ← 0
FOR Counter ← 1 TO 10
INPUT Number
Total ← Total + Number
NEXT Counter
OUTPUT Total
Example 2: Total the marks of a class.
INPUT ClassSize
Total ← 0
FOR Counter ← 1 TO ClassSize
INPUT StudentMark
Total ← Total + StudentMark
NEXT Counter
OUTPUT Total

2. Counting

  • Counting means finding how many times something happens.
  • To write a counting program, you usually:
    • initialise the counter to 0
    • test each value or event
    • increase the counter by 1 when the condition is met
  • The counter is often updated using Count ← Count + 1.
  • Counting can also be used to count down, for example when stock is reduced after a sale.
Example 1: Count how many of 10 entered numbers are greater than 50.
Count ← 0
FOR Counter ← 1 TO 10
INPUT Number
IF Number > 50 THEN
Count ← Count + 1
ENDIF
NEXT Counter
OUTPUT Count
Example 2: Count down the last 10 seconds before a race starts.
FOR Counter ← 10 TO 1 STEP -1
OUTPUT Counter
NEXT Counter
OUTPUT "Start"

3. Finding maximum and minimum values

  • Before comparing values, the maximum and minimum must be initialised first, otherwise the program may produce incorrect results.
  • If the valid range is known, you can initialise the maximum to a small value and the minimum to a large value.
  • If the valid range is not known, initialise both values from the first input, then continue from the second input.
  • Each new value is compared with the current maximum and minimum.
Example 1: Find the highest and lowest mark in a class when the range is known.
INPUT ClassSize
MaximumMark ← 0
MinimumMark ← 100
FOR Counter ← 1 TO ClassSize
INPUT StudentMark
IF StudentMark > MaximumMark THEN
MaximumMark ← StudentMark
ENDIF
IF StudentMark < MinimumMark THEN
MinimumMark ← StudentMark
ENDIF
NEXT Counter
Example 2: Find the highest and lowest values when the range is not known.
INPUT Number
Maximum ← Number
Minimum ← Number
FOR Counter ← 2 TO 10
INPUT Number
IF Number > Maximum THEN
Maximum ← Number
ENDIF
IF Number < Minimum THEN
Minimum ← Number
ENDIF
NEXT Counter

4. Finding average values

  • The average here means the mean.
  • The mean is found by adding all the values together, then dividing by how many values there are.
  • To find an average, a program needs a total and a count.
  • The formula is Average ← Total / Count.
Example 1: Find the average of 10 entered numbers.
Total ← 0
Count ← 0
FOR Counter ← 1 TO 10
INPUT Number
Total ← Total + Number
Count ← Count + 1
NEXT Counter
Average ← Total / Count
OUTPUT Average
Example 2: Find the average mark for a class.
INPUT ClassSize
Total ← 0
FOR Counter ← 1 TO ClassSize
INPUT StudentMark
Total ← Total + StudentMark
NEXT Counter
Average ← Total / ClassSize
OUTPUT Average
