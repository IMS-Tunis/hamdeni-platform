Exercise 6 - Correct the average algorithm

The algorithm should calculate the average of 6 numbers. Identify the logical error and write a corrected version.

P ← 0
Q ← 0
FOR R ← 1 TO 6
  INPUT S
  P ← P + S
NEXT R
T ← P / Q
OUTPUT T
Mark scheme
The value of Q is never updated.

Correct version:
P ← 0
Q ← 0
FOR R ← 1 TO 6
  INPUT S
  P ← P + S
  Q ← Q + 1
NEXT R
T ← P / Q
OUTPUT T

Exercise 8 - Write pseudocode

Write pseudocode to input 10 integers and count how many are positive and how many are negative. Zero is not counted as positive or negative.

Mark scheme
PositiveCount ← 0
NegativeCount ← 0
FOR Counter ← 1 TO 10
  INPUT Number
  IF Number > 0 THEN
    PositiveCount ← PositiveCount + 1
  ENDIF
  IF Number < 0 THEN
    NegativeCount ← NegativeCount + 1
  ENDIF
NEXT Counter
OUTPUT PositiveCount
OUTPUT NegativeCount

Exercise 9 - Write pseudocode

Write pseudocode to input 8 positive integers and calculate the average of only the even numbers. You may assume that at least one even number will be entered.

Mark scheme
TotalEven ← 0
CountEven ← 0
FOR Counter ← 1 TO 8
  INPUT Number
  IF Number MOD 2 = 0 THEN
    TotalEven ← TotalEven + Number
    CountEven ← CountEven + 1
  ENDIF
NEXT Counter
Average ← TotalEven / CountEven
OUTPUT Average
