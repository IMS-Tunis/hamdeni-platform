1. FOR loop

  • A count-controlled loop repeats a fixed number of times. It uses the structure FOR … TO … NEXT with a loop counter that changes each pass until the final value is reached.
    • The counter takes each integer value from the start to the end inclusive; the loop ends after executing at the end value.
    • Common patterns include FOR X ← 0 TO 9 … NEXT X and FOR Counter ← 0 TO 10 … NEXT Counter.
    Example: FOR loop that executes a statement ten times.
    FOR Counter ← 1 TO 10
    OUTPUT "*"
    NEXT Counter
    Example: FOR loop that uses the counter in a calculation on each pass.
    FOR Counter ← 1 TO 10
    OUTPUT Counter * Counter
    NEXT Counter
  • Use FOR … TO … NEXT when the number of repetitions is known in advance.

4. The use of STEP

  • STEP changes the counter movement. The counter takes values from the start to the end using the given increment. If the next value would pass the end, the loop terminates.
  • Termination rules: positive STEP runs upward until the end is reached or passed; negative STEP runs downward until the end is reached or passed.
Example: Odd numbers from 1 to 9 using STEP 2
FOR i ← 1 TO 9 STEP 2
OUTPUT i
NEXT i

Outputs: 1, 3, 5, 7, 9.

Example: Countdown in twos from 10 to 0
FOR n ← 10 TO 0 STEP -2
OUTPUT n
NEXT n

Outputs: 10, 8, 6, 4, 2, 0.

Example: STEP that does not land exactly on the end
FOR j ← 1 TO 10 STEP 4
OUTPUT j
NEXT j

Outputs: 1, 5, 9.

4. Nested For Loop

  • A nested FOR loop is one FOR … TO … NEXT inside another. The inner loop completes all of its iterations for every single iteration of the outer loop.
    Example: Generate a 3×4 multiplication grid (row × column).
    FOR Row ← 1 TO 3
    FOR Col ← 1 TO 4
    OUTPUT Row * Col
    NEXT Col
    NEXT Row
    Example: Display a 3×5 block of asterisks using a nested loop.
    FOR Row ← 1 TO 3
    FOR Col ← 1 TO 5
    OUTPUT "*"
    NEXT Col
    NEXT Row
