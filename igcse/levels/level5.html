1. WHILE loop

  • A WHILE loop is a pre-condition loop that tests the condition before the first iteration; it repeats while the condition is TRUE and stops when the condition becomes FALSE.
    • If the condition is FALSE at the start, the statements inside the loop do not run.
    • Cambridge pseudocode structure: WHILE condition DO … ENDWHILE.
    • The test for exiting the loop is at the beginning of the loop.

    This example shows a WHILE loop that outputs ten stars using a counter.

    Counter ← 0
    WHILE Counter < 10 DO
    OUTPUT "*"
    Counter ← Counter + 1
    ENDWHILE

    This example shows a WHILE loop using a counter to output 1 to 10.

    Number <- 1
    WHILE Number < 11 DO
    OUTPUT(Number)
    Number <- Number + 1
    ENDWHILE

2. REPEAT…UNTIL loop

  • A REPEAT…UNTIL loop is a post-condition loop that runs its statements once and then checks the condition at the end; it repeats until the condition becomes TRUE (that is, it loops while the condition is FALSE).
    • Cambridge pseudocode structure: REPEAT … UNTIL Condition.
    • The actions in this loop are always completed at least once.
    • The test for exiting the loop is at the end of the loop.

    This example shows a REPEAT … UNTIL loop that outputs ten stars using a counter.

    Counter ← 0
    REPEAT
    OUTPUT "*"
    Counter ← Counter + 1
    UNTIL Counter > 9

    This example shows a REPEAT…UNTIL loop that continues asking until the user enters "Yes".

    REPEAT
    OUTPUT ("Do you want to stop?")
    INPUT Answer
    UNTIL Answer = "Yes"
