6.3 Artificial Intelligence

Computer Science IGCSE notes, prepared by Dr. Hamdeni

6.3a: Understand what is meant by artificial intelligence

  • Artificial intelligence (AI) is used to simulate intelligent human behavior.
  • AI simulates Cognitive functions.
  • AI is evaluated based on how well it performs tasks that are normally carried out by humans, such as speaking, recognizing images, or solving problems.
  • AI systems can be categorized into three types:
    • Narrow AI
    • General AI
    • Strong AI
  • Simulated intelligence is the core idea behind AI.
  • Turing Test checks whether a human can distinguish between responses from a person and an AI.
  • AI is often described as a collection of Rules, Data, and the ability to reason, learn, and adapt to its environment.
  • Natural language systems can answer differently phrased questions like “When does the sun set today?” or “What time is sunset?” by identifying the intended meaning and responding accurately.
  • Smart home devices like Alexa or Siri use Speech recognition to take commands, learn user preferences, and improve future responses.
  • Chatbots simulate conversation by analyzing user input and generating appropriate replies, allowing them to interact with users in real time.
  • Facial expression recognition works by detecting the positions of eyebrows, eyes, and mouth to determine emotions like joy, anger, or surprise.
  • AI systems in journalism can generate news reports automatically by processing live data feeds. This allows real-time reporting without human intervention.
  • Imagine teaching your toaster to recognize burnt toast and avoid it next time. That’s AI trying to “think like you” in the kitchen.
  • Chatbots are like those friends who always text back instantly. These bots never sleep and reply to everyone.
  • The Turing Test is like a masked talent show. If the audience can’t tell whether the performer is a human or a robot, the AI wins.

6.3b: Describe the main characteristics of AI

  • AI systems often share four key features that support intelligent decision-making:
    • Data collection
    • Rules base
    • Reasoning
    • Learning and adaptation
  • Machine learning allows AI systems to adapt over time.
    • Machine learning involves training a system with data so it can recognize patterns and make decisions.
    • There are two main types of machine learning:
    • Supervised learning
    • Unsupervised learning
    • Clustering is commonly used to reveal hidden structure in the data.
  • Collaborative filtering allows the AI to make suggestions even without direct input from the user.
  • An expert system contains three essential components: the Knowledge base, the Rule base, and the Inference engine. It also includes a User interface.
  • A robot with machine learning capabilities can adjust its internal rules, data, or processes based on past experience.
  • A robot using machine learning becomes more efficient over time because it:
    • Learns from its environment (for example, by remembering where obstacles are)
    • Can recall where to begin and end tasks like planting seeds
    • Reduces its error rate by applying what it has learned from earlier actions
  • AI systems can be summarized by three key capabilities:
    • Collect data and use rules to make decisions
    • Use logic and reasoning to develop new conclusions
    • Learn and adapt their behavior over time
  • AI simulates Cognitive functions, enabling machines to operate intelligently.
  • A robot using Infra-red sensors might collect data and follow a rule like: "If an obstacle is detected, turn right."
  • An AI might apply logical reasoning: “All dogs eat meat. Fred is a dog. Therefore, Fred eats meat.” This shows how the AI uses known rules to make new conclusions.
  • In Spam filtering, the system learns from labeled examples and flags future messages containing common spam keywords such as “lottery” or “earn.”
  • In Product recommendation, the system compares your shopping behavior with others to recommend similar products.
  • Fraud detection works by analyzing spending patterns and flagging unusual behavior, such as an expensive jewelry purchase.
  • A search engine improves its ranking system using machine learning. If most users click a result on page 1, it is marked as useful and ranked higher in the future.
  • Think of machine learning like a kid learning to ride a bike. The more it falls, the better it gets!
  • Supervised learning is like teaching a dog tricks with treats. Unsupervised learning is your dog figuring out how to open the fridge on its own!
  • AI collecting data is like a robot keeping a diary — it remembers everything it senses to improve its decisions later.

6.3c: Explain the basic operation and components of AI systems

  • An Expert system helps users reach conclusions based on a structured set of rules and facts.
  • Expert systems simulate a real expert’s decision-making process by asking a series of questions and guiding the user toward a solution.
  • The main components of an expert system are:
    • Knowledge base
    • Rules base
    • Inference engine
    • User interface
  • Some expert systems include an Explanation system.
  • An expert system works by following these steps:
    • Taking input (such as symptoms or user observations) through the interface
    • Using the inference engine to compare user input with known facts in the knowledge base
    • Applying logic from the rules base to reach a decision or diagnosis
    • Presenting results, advice, or probabilities to the user
    • Optionally asking more questions to refine the outcome
  • The Inference engine:
    • Applies IF-THEN rules to known data
    • Draws conclusions or diagnoses based on reasoning
    • Decides if more input is needed to improve the result
  • A complete expert system also:
    • Uses the interface to receive input and display results
    • Combines logic from the rules base with facts from the knowledge base
    • Can decide the most relevant next question to ask the user
  • Advantages of expert systems include:
    • High level of accuracy and consistency
    • Fast response times compared to human experts
    • Can explain reasoning and give traceable logic
    • Can store and access vast amounts of information
    • Offer unbiased decisions based purely on logic
  • Disadvantages of expert systems include:
    • Require considerable user training to operate properly
    • Setup and maintenance costs are high
    • Responses may feel impersonal or “cold,” especially in sensitive contexts like medical care
    • The system’s performance depends entirely on the quality of its input data
    • Users may falsely assume the system is infallible and accept incorrect advice
  • Expert systems are also used in:
    • Oil and mineral exploration
    • Tax calculation and financial advising
    • Fault detection in electronics
    • Strategy games such as chess
    • Identifying animals, plants, or chemicals
  • A Machine learning system can adapt without needing to be reprogrammed.
    • In a Medical expert system, the system asks about symptoms, searches its knowledge base, and provides a possible diagnosis with a confidence percentage. For example: “Based on symptoms, there is a 21% probability of mercury poisoning.”
    • A printer troubleshooting system might ask: “Is the green light on?” If not, it recommends turning on the power. If yes, it moves to a different question based on this response.
    • A rule in the rules base might be: IF continent = “South America” AND language = “Portuguese” THEN country = “Brazil”.
    • In a reasoning task, if a system is given the facts: “It is a mammal. It is not a pet. It lives in water. It makes sonic sounds. It has skin. It has no legs,” it will identify the object as a whale.
    • An expert system is like a super-serious quiz master — it always knows the next question to ask based on your answers.
    • The inference engine is the “AI Sherlock Holmes” — it pieces together clues and draws logical conclusions.
