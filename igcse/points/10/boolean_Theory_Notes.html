Section 1

Boolean Foundations

Why Boolean logic?

Computers use electrical signals to represent binary data. A high signal represents 1 and a low signal represents 0. Boolean logic works with these two states, often described as true and false.

What is a logic gate?

A logic gate is a small electronic circuit that receives one or more binary inputs, applies a rule, and produces exactly one binary output. Several connected gates form a logic circuit.

Syllabus boundaries: NOT has one input. AND, OR, NAND, NOR and XOR have two inputs. Exam circuits have a maximum of three inputs and one output, and must be drawn exactly as stated without simplification.

Section 2

Logic Gates and Truth Tables

NOT

Reverses the input. 0 becomes 1 and 1 becomes 0.

X = NOT A

AND

Outputs 1 only when both inputs are 1.

X = A AND B

OR

Outputs 1 when either or both inputs are 1.

X = A OR B

NAND

The opposite of AND. Outputs 0 only for inputs 1 and 1.

X = A NAND B

NOR

The opposite of OR. Outputs 1 only for inputs 0 and 0.

X = A NOR B

XOR (EOR)

Outputs 1 when the two inputs are different.

X = A XOR B
Symbol clue: a small circle, called a bubble, means the result is inverted. NAND is AND followed by NOT; NOR is OR followed by NOT. XOR looks like OR with one extra curved line on the input side.

A logic circuit is read from left to right. Work out the output of each gate in order and record intermediate results in extra truth-table columns.

  1. List inputs
    Use binary order: 000, 001, 010, 011, 100, 101, 110, 111.
  2. Label stages
    Add D, E or other intermediate labels to the circuit and table.
  3. Work left to right
    Complete the earliest gates before later gates.
  4. Calculate output
    Use the final gate to complete X, Y or Z.
Wire rule: crossing wires with a dot are connected. Crossing wires without a dot are not connected. If one input is used more than once, draw it once and split its wire at a junction dot.

Section 4

Converting Representations

Expression → circuit

  1. Find the operator outside all brackets; it is the final gate.
  2. Draw each innermost bracket first.
  3. Feed those intermediate outputs toward the final gate.
  4. Do not simplify the stated expression.

Circuit → expression

  1. Begin with the output label and an equals sign.
  2. Identify the final gate nearest the output.
  3. Write the expressions for the branches feeding it.
  4. Use brackets so each gate operation is unambiguous.

Problem statement → expression

An alarm X sounds if A is ON and B is OFF, or if B is ON and C is OFF.

X = (A AND NOT B) OR (B AND NOT C)

Translate ON as 1, OFF as 0, and look for the keywords AND, OR and NOT. Each condition becomes a bracketed branch.

Truth table → expression: sum of products

If no single gate matches the table, focus only on rows where the output is 1. Inside each selected row, write a 1 input as its letter and a 0 input as NOT that letter. Join inputs with AND, then join the row expressions with OR.

Exam checklist

Use the correct standard gate symbols.
Include every input combination.
Use 2n rows for n inputs.
Show useful intermediate columns.
Use brackets clearly in expressions.
Draw the stated circuit without simplification.
Check that the final column answers the problem.
Remember XOR means inputs are different.
