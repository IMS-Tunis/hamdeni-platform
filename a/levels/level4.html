1. For Loops

  • A for loop repeats a block once for each item in a sequence such as a list, string, or a range of numbers.
Example 1: Iterating over a range of numbers
for i in range(5):
print(i)
Example 2: Counting down with a step value
for i in range(10, 0, -2):
print(i)
Example 3: Iterating over a list
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
print(fruit)

2. While Loops

  • A while loop repeats as long as its condition remains true.
Example 1: Simple counter
count = 1
while count <= 5:
print(count)
count += 1
Example 2: Sum numbers until a limit
total = 0
number = 1
while number <= 5:
total += number
number += 1
print("Sum:", total)

3. Control Flow in Loops

  • break stops the nearest loop immediately.
  • Tip: use for when the number of iterations is known; use while when you loop until a condition is met.
Example: Sentinel-controlled loop with break
while True:
number = int(input("Enter a number (0 to quit): "))
if number == 0:
break
print("You entered:", number)
