1. Writing Simple if Statements

  • An if statement executes a block only when its condition is true.
Example 1: Check voting eligibility
age = 18
if age >= 18:
print("You are eligible to vote.")
Basic if only

2. Understanding the Syntax and Indentation Rules for if Statements

  • Indentation defines the block controlled by if. Inconsistent indentation raises IndentationError.
Example 2: Indented block
if True:
print("This is indented.")
Indentation in if and if-else

3. Using else for Alternative Execution Paths

  • else provides a block that runs when the if condition is false.
Example 3: if-else decision
age = 16
if age >= 18:
print("You are eligible to vote.")
else:
print("You are not eligible to vote.")
Else only

4. Using elif for Multiple Conditions

  • elif checks the next condition when previous conditions are false. Only the first true branch runs.
Example 4: Grading with elif
score = 85
if score >= 90:
print("Grade: A")
elif score >= 80:
print("Grade: B")
else:
print("Grade: C")
Elif only

5. Nesting and Combining Conditional Statements

  • Nesting places one if inside another. Conditions can also be combined using and, or, and not.
Example 5: Nested if with a second condition
age = 20
citizenship = "Yes"
if age >= 18:
if citizenship == "Yes":
print("You are eligible to vote.")
else:
print("You need to be a citizen to vote.")
else:
print("You are not old enough to vote.")
