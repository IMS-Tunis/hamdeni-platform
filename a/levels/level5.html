1. Basic List Operations

  • Declaring an empty list: uses [] to represent no elements.
Example A - Declaring an empty list
my_list = [] # Creates an empty list
  • Declaring a list with a specific size: multiply a one-element list to repeat a default value.
Example B - Declaring a fixed-size list with default values
my_list = [None] * 10 # Creates a list with 10 None values
  • Initializing with specific values: put the values directly between brackets.
Example C - Initialize with given values
my_list = [1, 2, 3, 4, 5] # Creates a list with values 1 to 5
  • Appending to the end: append(x) adds a single element at the tail.
Example D - Append to end
my_list = [1, 2, 3]
my_list.append(6) # [1, 2, 3, 6]
  • Inserting at a position: insert(index, value) places an element before the given index.
Example E - Insert at a specific index
my_list = [1, 2, 3, 4]
my_list.insert(2, 99) # [1, 2, 99, 3, 4]
  • Removing values: remove(x) deletes the first occurrence of x.
Example F - Remove first occurrence
my_list = [1, 2, 3, 2, 4]
my_list.remove(2) # [1, 3, 2, 4]
  • Removing by index: del list[i] deletes without returning; pop(i) deletes and returns.
Example G - Remove by index using del and pop
my_list = [1, 3, 2, 4]
del my_list[1] # removes item at index 1 -> [1, 2, 4]
my_list = [1, 3, 2, 4]
popped_element = my_list.pop(2) # removes and returns element at index 2
# my_list becomes [1, 3, 4]; popped_element == 2
  • Combining lists: + concatenates two lists into a new one.
Example H - Concatenate two lists
list1 = [1, 2, 3]
list2 = [4, 5, 6]
combined_list = list1 + list2 # [1, 2, 3, 4, 5, 6]

2. Traversing and Modifying Lists

  • For-each traversal: the loop variable takes each element directly.
Example A - Iterate over elements
my_list = [1, 2, 3, 4, 5]
for item in my_list:
print(item)
  • Index-based traversal: use range(len(list)) to access by position.
Example B - Index-based traversal
my_list = [1, 2, 3, 4, 5]
for i in range(len(my_list)):
print(my_list[i])
  • Updating an element: assign a new value at a given index.
Example C - Update by index
my_list = [1, 2, 3, 4]
my_list[2] = 10 # [1, 2, 10, 4]

3. Working with Two-Dimensional Lists (2D Lists)

  • Declaring a 2D list: a list of lists, often used for matrices or grids.
Example A - Create a 3x3 zero matrix
rows, cols = 3, 3
matrix = [[0 for _ in range(cols)] for _ in range(rows)] # 3x3 matrix of zeros
  • Accessing and modifying: use matrix[row][col].
Example B - Access and modify an element
element = matrix[1][2] # access row 1, col 2
matrix[1][2] = 5 # update that position to 5
  • Traversing a 2D list: nested loops visit each element row by row.
Example C - Traverse rows and columns
for row in matrix:
for element in row:
print(element)

4. List Methods and Functions

  • len: number of items in a list or number of rows/columns in a 2D list.
Example A - len on lists and 2D lists
my_list = [10, 20, 30]
length = len(my_list) # 3
matrix = [[1,2,3],[4,5,6],[7,8,9]]
num_rows = len(matrix) # 3
num_columns = len(matrix[0]) # 3
  • index: returns the position of the first matching value.
Example B - index to find first occurrence
my_list = [10, 20, 30, 20]
index = my_list.index(20) # -> 1
# my_list.index(3) would raise ValueError
  • Membership with in: checks if a value exists in the list.
Example C - Membership test
my_list = [1, 3, 5, 7, 9]
if 5 in my_list:
print("5 is in the list")
  • Negative indices: count from the end, where -1 is the last element.
Example D - Negative indices
my_list = [10, 20, 30, 40]
last_element = my_list[-1] # 40
  • Slicing: extract a sublist using [start:end]; end is exclusive.
Example E - Slicing basics
my_list = [0, 1, 2, 3, 4, 5]
sub_list = my_list[1:4] # [1, 2, 3]
word = "123456789"
print(word[0:1]) # "1"
  • Omitted bounds: leaving start or end empty means from start or to end.
Example F - Slicing with omitted bounds
my_list = [0, 1, 2, 3, 4, 5]
start_from_second = my_list[1:] # [1, 2, 3, 4, 5]
up_to_third = my_list[:3] # [0, 1, 2]
full_list = my_list[:] # [0, 1, 2, 3, 4, 5]
