1. Overview of Python

  • Python is a high-level, interpreted programming language known for its simplicity and readability.
  • It is widely used for web development, data analysis, artificial intelligence, scientific computing, and many other applications.
  • Python's syntax emphasizes readability and allows programmers to express concepts in fewer lines of code compared to languages like C++ or Java.

2. Installing Python

To start programming in Python, you need to install Python on your computer. Follow these steps to install Python:

  1. Download the Installer: Visit the official Python website at python.org and download the latest version of Python for your operating system (Windows, macOS, or Linux).
  2. Run the Installer: Open the downloaded file and follow the installation instructions. Make sure to check the option to add Python to your system PATH.
  3. Verify Installation: Open a command prompt (Windows) and type python --version to verify the installation. You should see the version of Python you installed.

3. Setting Up a Development Environment

A good development environment helps you write and manage your code efficiently. You can use text editors like Notepad++ or more advanced Integrated Development Environments (IDEs) like PyCharm, Visual Studio Code, or Jupyter Notebook.

  • PyCharm: A powerful IDE specifically for Python development. It offers many features such as code completion, debugging, and project management.
  • Visual Studio Code: A lightweight but powerful source code editor that supports Python through extensions.
  • Jupyter Notebook: An open-source web application that allows you to create and share documents containing live code, equations, visualizations, and narrative text.

4. Writing Your First Python Program

Let's write and run a simple Python program that prints "Hello, World!" to the console. This will help you understand the basic structure of a Python script.

  1. Open your text editor or IDE.
  2. Write the following code:
Example 1: Hello World
print("Hello, World!")
  1. Save the file with a .py extension, for example, hello.py.
  2. Run the program: If you are using a command prompt or terminal, navigate to the directory where you saved the file and type python hello.py.

You should see the output:

Hello, World!
Hello World in Python (Short)

5. Basic Syntax and Structure

Python programs are written as plain text and saved with a .py extension. Here are some key points about Python syntax and structure:

  • Indentation: Python uses indentation to define blocks of code. Indentation must be consistent throughout your code.
  • Comments: Use the # symbol to add comments to your code. Comments are ignored by the interpreter.
Example 2: Comments
# This is a comment
print("Hello, World!") # This is an inline comment
  • Variables: Variables store data. Python is dynamically typed. Use descriptive names that start with a letter or underscore. Avoid starting with a digit or other symbols.
Example 3: Variables
message = "Hello, World!"
print(message)
  • Basic Data Types: Common data types in Python include integers, floats, strings, and booleans.
Example 4: Basic Data Types
age = 25 # Integer
price = 19.99 # Float
name = "Alice" # String
is_student = True # Boolean
