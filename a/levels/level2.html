1. Basic I/O

  • Reading user input: input() captures text typed in the console. Its return value is always a string.
Example 1: Read a name and greet
name = input("Enter your name: ")
print("Hello,", name)
  • Converting input to numbers: use int() or float() when arithmetic is needed.
Example 2: Convert input to integer
age = int(input("Enter your age: "))
print("You are", age, "years old.")

Why convert input types? input() always returns a string, while arithmetic needs numeric types such as int or float.

2. Operators

  • Arithmetic operators: +, -, *, /, % (modulus), ** (power), // (floor division).
Example A: Arithmetic operators
a = 10
b = 3
print("Addition:", a + b)
print("Subtraction:", a - b)
print("Multiplication:", a * b)
print("Division:", a / b)
print("Modulus:", a % b)
print("Exponentiation:", a ** b)
print("Floor Division:", a // b)

What is modulus? The operator % returns the remainder of a division, e.g., 10 % 3 is 1.

Division vs floor division: / returns a float; // returns the largest integer ≤ the exact quotient.

  • Operator precedence: multiplication and division happen before addition and subtraction unless parentheses change the order.
Example B: Operator precedence
result = 10 + 5 * 2
print(result)
result = (10 + 5) * 2
print(result)
  • Comparison operators: test relationships and return booleans.
Example C: Comparison operators
a = 10
b = 5
print(a == b) # False
print(a != b) # True
print(a > b) # True
print(a < b) # False
print(a >= b) # True
print(a <= b) # False
  • Logical operators: combine booleans to form complex conditions.
Example D: Logical operators
x = True
y = False
print(x and y) # False
print(x or y) # True
print(not x) # False

Why use logical operators? They allow multiple conditions in one expression so programs can make richer decisions.

  • Combining multiple conditions: use and and or for compound tests.
Example E: Combining conditions
age = 25
is_student = True
print(age > 18 and is_student) # True
print(age < 18 or is_student) # True
